# 使用C++编写的多线程HTTP下载器

此项目为玩具项目，主要用于练手。项目目标：在尽量不使用第三方库的情况下支持以下功能：

- [x] HTTP下载
- [x] HTTPS下载（基于OpenSSL）
- [x] 多线程下载
- [ ] 处理HTTP重定向状态码
- [x] 支持Linux平台
- [x] 支持Windows平台
